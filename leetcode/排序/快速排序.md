
快速排序就是随机选一个数，把小于它的都放到左边，大于它的都放到右边

## 1. 怎么随机选

go 中随机数是`rand.Intn(n)`生成`[0,n)`范围的随机数

所以在`[left, right]`中随机选一个数，就是
```go
idx := left + rand.Intn(right - left + 1)
```

## 2.怎么把小于它的都放到左边，大于它的都放到右边

答案是**双指针**

先把要放的数弄到最左边
```go
nums[l], nums[idx] = nums[idx], nums[l]
```
双指针 i，j，i 左边的都是比 pivot 小的，j 右边的都是比 pivot 大的，如果满足这个条件就不断递增递减，直到不满足，此时 i，j 需换位，因为他们满足了另一边的要求，然后再递增递减一次

![[aaab7d2e6e5c54c6dcc59c945f11e358.jpg]]
```go
i := l + 1
j := r
for {
	for i <= j && nums[i] < pivot {
		i++
	}
	for i <= j && nums[j] > pivot {
		j--	
	}
	if i > j {
		break
	}

	nums[i], nums[j] = nums[j], nums[i]
	i++
	j--

}
nums[l], nums[j] = nums[j], nums[l]
```
