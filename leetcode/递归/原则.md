1. 确定递归函数的意义
2. 从最后一步开始推，大胆相信递归函数，应用迭代
3. return 的值是函数参数的值
4. 找终止条件

若想记录递归过程中出现的值，则该变量必须在递归嵌套外面（eg. res）
```python
		res = []
        def dfs(node):
            if node is None:
                return
            res.append(node.val)
            dfs(node.left)
            dfs(node.right)
        dfs(root)
        return res
```
