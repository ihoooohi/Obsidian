怎么手写malloc呢，找到符合条件的空间给变量用

什么是对齐

就是malloc 返回的指针，**必须能安全存放任何类型**
```c
void *p = malloc(100);

(int*)p;        // OK
(double*)p;     // OK
(struct S*)p;   // OK

```

指针类型在 64 位机器上是 8 字节，在 32 位机器上是 4 字节。
如果一台机器：

- 地址是 32 位 → 最多能表示 2322^{32}232 个字节 → 指针要 **4 字节**
    
- 地址是 64 位 → 最多能表示 2642^{64}264 个字节 → 指针要 **8 字节**
    

👉 **指针必须“刚好能装下一个地址”**
