
# 1.Bit Stuffing.

## **(a)**

**Given data:**

```text
10011111101001111100011
```

Insert `0` after every five consecutive `1`s  

```text
100111110010011111000011
```

**Add flags (01111110):**

```text
0111111010011111001001111100001101111110
```

**Answer:**

```text
0111111010011111001001111100001101111110
```


## **(b)**

**Received frame:**

```text
111111001111101010111110011111111
```

**Remove flag (01111110):**

```text
11111101111111
```

Remove `0` after every five consecutive `1`s  

```text
1111111111111
```

**Answer:**

```text
1111111111111
```
