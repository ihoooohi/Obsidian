轻量级实时聊天与语音平台

Spring Boot / WebSocket / Redis / RabbitMQ / pgsql

基于 Spring Boot + WebSocket 实现实时聊天系统，支持频道聊天、成员状态同步和在线用户管理

设计用户、频道、消息等核心数据模型，使用 pgsql 持久化聊天记录并支持历史消息查询

使用 Redis 缓存用户在线状态、频道成员列表和未读消息数，减少数据库访问并提升系统响应速度

引入 RabbitMQ 实现消息异步落库和频道事件广播，提高系统吞吐能力并降低主链路延迟

实现 JWT 鉴权、统一异常处理与接口规范化设计，提升系统安全性与可维护性
