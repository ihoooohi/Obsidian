前端用Tailwind CSS组件库
前端用 react + vite 构建

数据库用 pgsql，远程的，在环境变量配置数据库的端口和账号密码等
```text
# PostgreSQL connection string (used only when DB_DRIVER=postgres; overrides PG_* below if set)
DB_DSN=
# PostgreSQL split settings (used when DB_DSN is empty)
PG_HOST=localhost
PG_PORT=5432
PG_USER=postgres
PG_PASSWORD=postgres
PG_DATABASE=echocenter
PG_SSLMODE=disable
```
