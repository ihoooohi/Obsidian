# 登录验证模块

```
┌─────────────────────────────────────────────────────────────────┐
│                        用户登录                                  │
└──────────────────────────┬──────────────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────────────┐
│              1. 验证用户名密码 (bcrypt)                          │
└──────────────────────────┬──────────────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────────────┐
│              2. 生成JWT Token                                    │
│              (AccountID + Username + 24h过期)                   │
└──────────────────────────┬──────────────────────────────────────┘
                           │
            ┌──────────────┴──────────────┐
            ▼                              ▼
     ┌─────────────┐                ┌─────────────┐
     │   MySQL    │                │   Redis    │
     │ account表  │                │account:{id}│
     │ Token字段  │                │  TTL:24h   │
     └─────────────┘                └─────────────┘
```
