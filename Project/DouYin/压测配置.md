##  数据集配置

默认数据集规模：

| 数据类型          | 数量        |
| ------------- | --------- |
| 用户 (users)    | 50,000    |
| 视频 (videos)   | 200,000   |
| 点赞 (likes)    | 2,000,000 |
| 评论 (comments) | 500,000   |
| 关注 (follows)  | 300,000   |

数据集通过 `run-benchmark.ps1` 脚本自动初始化到 MySQL 数据库中。

## 测试接口汇总

|接口|方法|适用场景|说明|
|---|---|---|---|
|`/feed/listLatest`|POST|S1/S2/S3|获取用户关注列表的最新视频 Feed 流|
|`/account/register`|POST|S2/S3 setup|用户注册（创建压测用的测试用户）|
|`/account/login`|POST|S2/S3 setup|用户登录（获取认证 token）|
|`/like/like`|POST|S2/S3|视频点赞接口|
|`/comment/publish`|POST|S2/S3|评论发布接口|
