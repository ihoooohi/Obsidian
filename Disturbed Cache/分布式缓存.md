分布式缓存分为分布式和缓存
## 分布式

什么是分布式？

多个节点（服务器）完成同一个系统目标，并且节点之间需要通信

怎么通信？

通信有三层：
```text
【调用方式】   RPC / REST
【传输协议】   HTTP / TCP
【序列化】     JSON / Protobuf
```
